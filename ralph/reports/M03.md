## Learnings
- The current repo does not include Tailwind or shadcn/ui setup, so the M03 shell was implemented with plain React + CSS while preserving the locked visual direction and future component boundaries.
- `apps/web/src/theme/tokens.ts` is now the design-system source for later frontend milestones; later work should extend those tokens rather than re-hardcoding colors or typography in new components.
- The responsive mobile navigation is stateful in `App.tsx`, which gives later milestones a stable place to attach spawn-wizard entry points and route-aware nav behavior without restructuring the header again.
